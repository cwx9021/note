案例1:为虚拟机 server 配置以下静态地址参数
– 主机名:server0.example.com
答：vim /etc/hoatname  
   i:server0.example.com :wq
– IP地址:172.25.0.120
– 子网掩码:255.255.255.0
– 默认网关:172.25.0.254
答：nmcli connection modify 'System eth0' ipv4.method manual ipv4.addresses '172.25.0.120/24 172.25.0.254' connection.autoconnect yes
    nmcli connection up 'System eth0'
– DNS服务器:172.25.254.254
答：vim /etc/resolv.conf
  i:nameserver 172.25.254.254  :wq


案例2:指定yum软件源
为 server0 指定可用的 yum 软件源
– YUM软件库的地址为 http://classroom.example.com/content/rhel7.0/x86_64/dvd
– 将此配置为虚拟机 server0 的默认软件仓库
答：vim  /etc/yum.repos.d/server0.repo
[server0]
name=server
enabled=1
gpgcheck=0
baseurl=http://classroom.example.com/content/rhel7.0/x86_64/dvd

– 确认可用的仓库列表
答：yum repolist
– 利用yum仓库安装system-config-kickstart 
答：yum -y install system-config-kickstart

案例4:tar制作/释放归档压缩包(zcf、ztf、zxf、jcf、jtf、jxf、cf、tf）
  首先创建/root/boothome/与/root/usrsbin/目录
答：mkdir /root/boothome/
   mkdir /root/usrsbin/
    1）备份/boot、/home这两个文件夹，保存为boothome.tar.gz文件
答：tar -zcf /root/boothome/boothome.tar.gz   /boot/ /home/

    2）查看boothome.tar.gz文件内包含哪些内容 
答：tar -tf /root/boothome.tar.gz 
    3）将boothome.tar.gz释放到文件夹/root/boothome/下
答： tar -xf /root/boothome.tar.gz -C /root/boothome/
    4）备份/usr/sbin目录，保存为usrsbin.tar.bz2文件
答：tar  -jcf /root/usrsbin/usrebin.tar.bz2   /usr/sbin/
    5）查看usrsbin.tar.bz2文件内包含哪些内容
答：tar -xf /root/usrsbin/usrebin.tar.bz2 

    6）将usrsbin.tar.bz2释放到/root/usrsbin/文件夹下
答：tar -jxf /root/usrsbin.tar.bz2  -C   /root/usrsbin

案例6：添加账号时设置不同属性
    1）新建用户nsd01，宿主目录位于/opt/nsd01
 答：useradd -d /opt/nsd01 nsd01
    2）新建系统账号sys01，将UID设为1234，登录Shell设为/sbin/nologin
 答：useradd -u 1234  -s /sbin/nologin sys01

    3）为用户sys01设置密码，并测试是否能够登录
 答：passwd sys01 123321  su - sys01
    4）新建用户admin，将其基本组设为users，附加组设为adm、root
 答：groupadd usres
    useradd -G usres admin
     gpasswd -a admin adm
     gpasswd -a admin root

案例10：配置NTP网络时间客户端
	配置虚拟机 server0，自动校对系统时间

	NTP服务器位于 classroom.example.com 
答：vim /etc/chrony.conf
#server 0.rhel.pool.ntp.org iburst
#server 1.rhel.pool.ntp.org iburst
#server 2.rhel.pool.ntp.org iburst
server classroom.example.com iburst

       此客户机的时间与NTP服务器的时间保持同步
systemctl restart chronyd  
systemctl enable chronyd
