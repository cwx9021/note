
案例：复制、删除、移动
  1. 在目录/opt下创建一个子目录 nsd 
    [root@jigege ~]# mkdir /opt/nsd
  2. 在目录/opt/nsd/创建文件readme.txt,利用vim写入内容 I Love Linux
    [root@jigege ~]# touch /opt/nsd/readme.txt
    [root@jigege ~]# vim /opt/nsd/readme.txt
     i love linux   Esc ---Shift+;    w q
  3. 将/etc/passwd 和 /etc/resolv.conf同时拷贝到/opt/nsd目录下
    [root@jigege ~]# cp -r /etc/passwd /etc/resolv.conf /opt/nsd

  4. 将文件 /etc/redhat-release复制到 /root/ 下，同时 改名为 version.txt 
    [root@jigege /]# cp /etc/redhat-release/ /root/version.txt
  5. 将文件 /root/version.txt 移动到/opt/nsd/目录下 
   [root@jigege /]# cp /root/version.txt /opt/nsd
  6. 将/home目录复制到/opt/nsd/目录下 
    [root@jigege ~]# cp -r /home/ /opt/nsd



案例：ls命令练习
  1. 查看根目录下内容
   [root@jigege /]# ls /
  2. 显示根目录本身详细属性
   [root@jigege /]# ls -ld /
  3. 切换到/boot目录，显示当前目录内容的详细属性，并加上易读的容量单位
   [root@jigege /]# cd /root
   [root@jigege ~]# ls -lh /root
  4. 显示/root的全部内容，包括隐藏文档
   [root@jigege ~]# ls -A /root

  5. 显示/bin/bash程序，详细属性
[root@jigege /]# ls -l /bin/bash
  6. 显示/dev/cdrom光驱设备，详细属性
[root@jigege /]# ls -l /dev/cdrom



案例：mount挂载
  1. 在根目录下创建目录结构/rhel7/dvd
  [root@jigege /]# mkdir -p /rhel7/ /dvd/
  2. 利用图形将光盘文件rhel-server-7.4-x86_64-dvd.iso放入光驱设备中

  3. 将光驱设备挂载到/rhel7/dvd目录，以/rhel7/dvd目录作为其访问点
[root@jigege /]# mount /dev/cdrom/ /rhel7/dvd/

  4. 查看/rhel7/dvd/Packages目录内容
   [root@jigege /]# ls /rhel7/dvd/Packages
  5. 将/rhel7/dvd/Packages目录中以vsftpd开头的软件包，拷贝到/opt下
[root@jigege /]#cp /rhel7/dvd/Packages/vsftpd* opt

案例：别名与vim的使用
  1. 为虚拟机定义一个别名，执行byebye可以实现关闭系统
  [root@jigege ~]# alias byebye='poweroff'
  2. 利用vim 修改文件/etc/hostname将其原有内容全部删除，写入新的内容为student.tmooc.cn
  [root@jigege ~]# vim /etc/hostname
