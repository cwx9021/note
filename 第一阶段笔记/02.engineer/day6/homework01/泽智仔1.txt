案例2:为虚拟机 server 配置Web访问控制
在 Web 网站 http://server0.example.com 的 DocumentRoot 目录下创建一个名为 private 的子目录，要求如下：
1）在server0.example.com的DocumentRoot目录下，private的子目录里书写网页文件index.html内容为  不许偷看丫
[root@server0 ~]# vim /etc/httpd/conf.d/nsd01.conf
[root@server0 ~]# mkdir /var/www/nsd01/private
[root@server0 ~]# echo '<h1>不许偷看丫' > /var/www/nsd01/private/index.html
[root@server0 ~]# cat /var/www/nsd01/private/index.html
<h1>不许偷看丫
[root@server0 ~]# firefox server0.example.com/private
[root@desktop0 ~]# firefox server0.example.com/private

2）此页面只能在本机浏览，但是从其他系统不能访问这个目录的内容
[root@server0 ~]# vim /etc/httpd/conf.d/nsd02.conf
[root@server0 ~]# cat /etc/httpd/conf.d/nsd02.conf
<Directory /var/www/nsd01/private>
 Require ip 172.25.0.11
</Directory>
[root@server0 ~]# systemctl restart httpd
[root@server0 ~]# firefox server0.example.com/private
[root@desktop0 ~]# firefox server0.example.com/private

案例4：为虚拟机 server 部署动态WSGI站点
为站点 webapp0.example.com 配置提供动态Web内容，要求如下：
1）此虚拟主机侦听在端口8909
2）测试网页从以下地址下载，不要作任何更改http://classroom/pub/materials/webinfo.wsgi 
3）从浏览器访问 http://webapp0.example.com:8909 可接收到动态生成的 Web 页面
[root@server0 ~]# cd /var/www/nsd02
[root@server0 nsd02]# wget http://classroom/pub/materials/webinfo.wsgi 
[root@server0 nsd02]# cat webinfo.wsgi 
[root@server0 nsd02]# vim /etc/httpd/conf.d/nsd01.conf
[root@server0 nsd02]# systemctl restart httpd
[root@server0 nsd02]# cd
[root@server0 ~]# systemctl restart httpd
[root@server0 ~]# firefox webapp0.example.com
[root@server0 ~]# yum -y install mod_wsgi
[root@server0 ~]# vim /etc/httpd/conf.d/nsd01.conf
[root@server0 ~]# cat /etc/httpd/conf.d/nsd01.conf
<VirtualHost *:80>
 ServerName server0.example.com
 DocumentRoot /webroot
</VirtualHost>
<VirtualHost *:80>
 ServerName www0.example.com
 DocumentRoot /var/www/nsd02
</VirtualHost>
<VirtualHost *:80>
 ServerName webapp0.example.com
 DocumentRoot /var/www/nsd03
WsgiScriptAlias / /var/www/nsd02/webinfo.wsgi
</VirtualHost>
[root@server0 ~]# systemctl restart httpd
[root@server0 ~]# firefox webapp0.example.com
[root@server0 ~]# vim /etc/httpd/conf.d/nsd01.conf
[root@server0 ~]# cat /etc/httpd/conf.d/nsd01.conf
<VirtualHost *:80>
 ServerName server0.example.com
 DocumentRoot /webroot
</VirtualHost>
<VirtualHost *:80>
 ServerName www0.example.com
 DocumentRoot /var/www/nsd02
</VirtualHost>
Listen 8909
<VirtualHost *:8909>
 ServerName webapp0.example.com
 DocumentRoot /var/www/nsd03
WsgiScriptAlias / /var/www/nsd02/webinfo.wsgi
</VirtualHost>
[root@server0 ~]# semanage port -a -t http_port_t  -p tcp  8909
[root@server0 ~]# systemctl restart httpd
[root@server0 ~]# firefox webapp0.example.com:8909
