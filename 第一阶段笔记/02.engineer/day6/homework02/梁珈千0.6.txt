案例3:为虚拟机 server 使用自定Web根目录

调整 Web 站点 http://server0.example.com 的网页目录，要求如下：
1）新建目录 /webroot，作为此站点新的网页目录
2）确保站点 http://server0.example.com  仍然可访问


[root@server0 ~]# mkdir /var/www/webroot
[root@server0 ~]# echo 'hahaha' > /var/www/webroot/index.html
[root@server0 ~]# cat /var/www/webroot/index.html
hahaha
[root@server0 ~]# vim /etc/httpd/conf/httpd.conf 
DocumentRoot "/var/www/webroot"
[root@server0 ~]# systemctl restart httpd



案例4：为虚拟机 server 部署动态WSGI站点
为站点 webapp0.example.com 配置提供动态Web内容，要求如下：
1）此虚拟主机侦听在端口8909
2）测试网页从以下地址下载，不要作任何更改http://classroom/pub/materials/webinfo.wsgi 
3）从浏览器访问 http://webapp0.example.com:8909 可接收到动态生成的 Web 页面

[root@server0 /]# mkdir /hahah
[root@server0 /]# chcon -R --reference=/var/www /hahah
[root@server0 /]# ls -Zd /hahah
[root@server0 /]# cd /hahah
[root@server0 hahah]# wget http://classroom/pub/materials/webinfo.wsgi 

vim /etc/httpd/conf.d/hahah.conf

<VirtualHost *:80>

  ServerName webapp0.example.com

  DocumentRoot  /var/www/hahah

  Alias   /   /var/www/hahah/webinfo.wsgi
 </VirtualHost>

  systemctl restart  httpd               

vim /etc/httpd/conf.d/hahah.conf
改为WsgiScriptAlias /  /var/www/hahah/webinfo.wsgi
