案例2:为虚拟机 server 配置Web访问控制
在 Web 网站 http://server0.example.com 的 DocumentRoot 目录下创建一个名为 private 的子目录，要求如下：
1）在server0.example.com的DocumentRoot目录下，private的子目录里书写网页文件index.html内容为  不许偷看丫
2）此页面只能在本机浏览，但是从其他系统不能访问这个目录的内容

	[root@server0 ~]# mkdir -p /var/www/fczlm4/private
	[root@server0 ~]# echo '<h1>不许偷看丫'> /var/www/fczlm4/private/index.html
	[root@server0 ~]# vim /etc/httpd/conf.d/acl.conf 
	<Directory "/var/www/fczlm4/private">
    	Require ip 172.25.0.11
	</Directory>
	[root@server0 ~]# systemctl restart httpd
	[root@desktop0 ~]# firefox server0.example.com/private





案例3:为虚拟机 server 使用自定Web根目录(chcon -R --reference=/var/www /webroot/)

调整 Web 站点 http://server0.example.com 的网页目录，要求如下：
1）新建目录 /webroot，作为此站点新的网页目录
2）确保站点 http://server0.example.com  仍然可访问

	[root@server0 ~]# mkdir /webroot
	[root@server0 ~]# echo '<h1>今天今天星闪闪' > /webroot/index.html
	[root@server0 ~]# vim /etc/httpd/conf.d/web.conf 
	<VirtualHost *:80>
	ServerName server0.example.com
	DocumentRoot /webroot
	[root@server0 ~]# vim /etc/httpd/conf.d/acl.conf
	<Directory "/webroot">
    	Require all granted
	</Directory>
	[root@server0 ~]# chcon -R --reference=/var/www /webroot/
	[root@server0 ~]# systemctl restart httpd
	[root@desktop0 ~]# firefox server0.example.com

	



案例4：为虚拟机 server 部署动态WSGI站点
为站点 webapp0.example.com 配置提供动态Web内容，要求如下：
1）此虚拟主机侦听在端口8909
2）测试网页从以下地址下载，不要作任何更改http://classroom/pub/materials/webinfo.wsgi 
3）从浏览器访问 http://webapp0.example.com:8909 可接收到动态生成的 Web 页面

	[root@server0 ~]# cd /var/www/38li
	[root@server0 38li]# wget http://classroom/pub/materials/webinfo.wsgi
	Alias / /var/www/38li/webinfo.wsgi
	[root@server0 38li]# systemctl restart httpd
	[root@desktop0 ~]# firefox webapp0.example.com
	[root@server0 /]# yum -y install mod_wsgi.x86_64
	[root@server0 /]# vim /etc/httpd/conf.d/web.conf
	<VirtualHost *:80>
	ServerName webapp0.example.com
	DocumentRoot /var/www/38li
	#Alias / /var/www/38li/webinfo.wsgi
	WsgiscriptAlias / /var/www/38li/webinfo.wsgi
	</VirtualHost>
	[root@server0 /]# systemctl restart httpd
	[root@desktop0 ~]# firefox webapp0.example.com
	Listen 8909
	<VirtualHost *:8909>
	ServerName webapp0.example.com
	DocumentRoot /var/www/38li
	#Alias / /var/www/38li/webinfo.wsgi
	WsgiscriptAlias / /var/www/38li/webinfo.wsgi
	</VirtualHost>
	[root@server0 /]# semanage port -l | grep http
	[root@server0 /]# semanage port -aWsgiscriptAlias / /var/www/nsd02/webinfo.wsgi -t http_port_t -p tcp 8909
	[root@server0 /]# systemctl restart httpd
	[root@desktop0 ~]# firefox webapp0.example.com:8909
